# Структура проекта Barkery Bot

## Основные файлы
- `barkery_bot.py` - Главный файл запуска бота
- `config.py` - Конфигурация (токены, настройки)
- `database.py` - Модели базы данных и подключение
- `handlers.py` - Обработчики команд и колбэков
- `admin.py` - Админ-панель
- `keyboards.py` - Inline клавиатуры
- `services.py` - Бизнес-логика (корзина, каталог, заказы)
- `notifications.py` - Уведомления для админа
- `setup_bot.py` - Настройка окружения

## Скрипты
- `start_bot.sh` - Запуск бота с проверками
- `manage_bot.sh` - Управление ботом (start/stop/status/logs)
- `scripts/backup_handlers.sh` - Создание резервных копий

## Вспомогательные файлы
- `requirements.txt` - Зависимости Python
- `.env` - Конфиденциальные настройки (не в репозитории)
- `.env.example` - Пример настроек
- `.gitignore` - Игнорируемые файлы

## Структура каталогов
- `backup/` - Резервные копии
  - `cleanup_20260130/` - Бэкап перед очисткой
  - `clean_version_backup_20260129_120224/` - Чистая версия
  - `old/` - Старые бэкапы
  - `single_files/` - Отдельные резервные файлы
  - `README.md` - Описание бэкапов
- `scripts/` - Вспомогательные скрипты
- `temp/` - Временные файлы
- `venv/` - Виртуальное окружение Python

## База данных
- `barkery.db` - SQLite база данных (создается автоматически)

## Функциональность
1. Интернет-магазин собачьих лакомств
2. Три неиерархические категории товаров
3. Весовые и штучные товары
4. Корзина с добавлением/удалением товаров
5. Оформление заказов с уведомлением админа
6. Админ-панель для управления товарами

## Запуск
1. `python3 setup_bot.py` - настройка
2. `./start_bot.sh` - запуск
3. `./manage_bot.sh start` - запуск в фоне
4. `./manage_bot.sh status` - проверка статуса
