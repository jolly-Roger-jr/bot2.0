# Резервные копии проекта Barkery Bot

## Структура
-  - старые бэкапы
-  - бэкап перед очисткой проекта

## Политика бэкапов
1. Храним не более 5 бэкапов каждого файла
2. Новые бэкапы создаются через скрипт: scripts/backup_handlers.sh
3. Автоматическое удаление старых бэкапов при превышении лимита

## Как создать бэкап
```bash
./scripts/backup_handlers.sh create
```

## Как восстановить
```bash
# Найдите нужный бэкап
ls -la backup/*.py

# Скопируйте его
cp backup/handlers_YYYYMMDD_HHMMSS.py handlers.py
```
